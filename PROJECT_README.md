# 《校园智行者：算法挑战赛》Web 版

## 项目概述

《校园智行者》是一个为中学生设计的交互式算法教学网站。该项目将抽象的计算机科学概念与学生的日常校园生活相结合，通过游戏化的关卡设计和实时的算法可视化，帮助学生理解和掌握核心编程算法。

## 核心特色

### 1. 三大精彩关卡

| 关卡名称 | 核心算法 | 学习目标 | 场景设定 |
| :--- | :--- | :--- | :--- |
| **图书馆整理** | 快速排序 (Quick Sort) | 掌握分治思想与递归 | 整理乱序的新书编号 |
| **晨读不迟到** | 广度优先搜索 (BFS) | 理解状态空间搜索 | 在校园地图中规划最短路径 |
| **社团活动达人** | 贪心算法 (Greedy) | 学习贪心策略与优化 | 在时间冲突中选择最多活动 |

### 2. 设计理念：现代教育科技风格

- **色彩方案**：深蓝色 (#1e40af) 与青绿色 (#0891b2) 的组合，传达学术严谨与创新活力
- **排版系统**：Poppins 字体用于标题（现代感），Inter 字体用于正文（易读性），Fira Code 用于代码展示
- **布局范式**：非对称设计，英雄区域 + 关卡卡片网格，充分利用视觉空间
- **交互体验**：流畅的动画过渡、即时的视觉反馈、渐进式的信息披露

### 3. 核心功能

**首页 (Home.tsx)**
- 英雄区域：展示游戏主视觉与核心价值主张
- 关卡介绍：三个关卡的卡片展示，包含算法名称、复杂度、难度等级
- 特性介绍：可视化演示、游戏化体验、学术严谨、完全免费
- 底部 CTA：引导用户开始挑战

**游戏页面 (Game.tsx)**
- 上方 60%：算法实时演示区，支持播放/暂停/重置
- 下方 40%：思考题与答题反馈区，强化学习效果
- 得分系统：每题 10 分，实时显示玩家进度

## 设计亮点

### 1. 可视化算法演示
每个关卡都提供实时的算法执行动画，使用颜色变化和元素移动来展示数据的变化过程。例如，在排序关卡中，被比较的元素会高亮显示，让学生直观看到算法的执行步骤。

### 2. 游戏化学习体验
- **关卡解锁**：学生需要完成前一个关卡才能进入下一个
- **得分系统**：正确回答思考题获得分数，激励学生深入思考
- **成就反馈**：完成关卡时显示庆祝动画和鼓励信息

### 3. 学术严谨性
每个关卡都包含：
- 算法的伪代码或代码实现
- 时间复杂度与空间复杂度分析
- 实际应用场景说明
- 思考题考察对核心概念的理解

### 4. 响应式设计
- 桌面端：充分利用屏幕空间，左右分栏布局
- 平板端：适配中等屏幕，优化触摸交互
- 手机端：单栏布局，优先展示关键内容

## 开发指南

### 本地开发
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:3000
```

### 构建生产版本
```bash
# 构建静态文件
pnpm build

# 预览生产版本
pnpm preview
```
